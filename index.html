Nivel inicial

Fundamentos de programación

Este curso introductorio te brinda las bases esenciales para comenzar tu camino como programador. Aprenderás los conceptos fundamentales de la programación, desde la lógica y la sintaxis hasta la resolución de problemas y la creación de tus primeros programas.

  • stacks

    8 Modulos

  • schedule

    24hs

  • terminal

    100 ejercicios

Al finalizar este curso, seras capaz de:

  • Dominar los conceptos básicos de la programación
  • Desarrollar habilidades de pensamiento crítico y analítico
  • Abrir las puertas a una amplia gama de oportunidades profesionales

Tecnologias que aprenderas:

  • Globstones
  • Scratch
  • MDN
  • Deepnote
  • GIT

Nivel Intermedio

Desarrollo web Front-End

Este curso te permitirá adquirir las habilidades necesarias para crear interfaces web atractivas e interactivas utilizando HTML, CSS y JavaScript. Domina las herramientas y metodologías para construir sitios web modernos y de alto rendimiento.

  • Ingresar a una industria en constante crecimiento.

Nivel Avanzado

Desarrollo web Back-End

Este curso está dirigido a desarrolladores front-end que buscan dar un paso adelante y dominar el back-end. Aprenderás los fundamentos del desarrollo del lado del servidor, incluyendo lenguajes como Python, Java o JavaScript, bases de datos y arquitecturas web escalables. Con estas habilidades, estarás preparado para crear aplicaciones web completas y robustas.

  • Implementar arquitecturas web escalables y de alto rendimiento.
